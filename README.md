# fp2w
Tired of reading & comparing 40 hex digits just to get some digital security?
This tool could help you!

## Dependencies
- gpg2
- bc
- awk, sed, tr (you should probably have these already)

## Usage
```bash
./fp2w your@email.xyz
```
